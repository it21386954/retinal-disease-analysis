Home

Welcome to the Retinal Disease Analysis System project. This research aims to develop a cutting-edge system for early detection and analysis of retinal diseases using advanced deep learning and machine learning techniques.

Our system integrates image classification, health record analysis, personalized treatment recommendations, and disease progression tracking to support healthcare professionals and improve patient outcomes.

The website provides an overview of our research, milestones, key documents, presentations, and the dedicated project team.

Project Components

  • Component 1: Retinal Disease Image Classification using Convolutional Neural Networks (CNN) for accurate image-based diagnosis.
  • Component 2: Retinal Disease Prediction by integrating vision test results with health records using machine learning.
  • Component 3: Personalized Treatment Recommendation based on patient response analysis.
  • Component 4: Disease Progression Analysis using LSTM and HOG models to forecast future disease states.
